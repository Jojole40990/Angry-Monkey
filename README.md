# Angry Monkey
A 2D survivor roguelite game built with Godot, inspired by Vampire Survivors

## Overview
Angry Monkey is a 2D top-down survivor game where the player fights off endless waves of enemies, collects XP, levels up, and chooses weapon upgrades to survive as long as possible. The game features escalating difficulty with boss fights and auto-attack combat.

## Features

- Auto-attack and auto-aim combat system
- XP collection and level-up progression
- Weapon and upgrade selection on level up
- Wave-based enemy spawning with increasing difficulty
- Boss fights at milestone waves

## Tech Stack

- Engine: Godot 4.3
- Language: GDScript

## Installation

- Download and install [Godot 4.3](https://godotengine.org/download)
- Clone the repository:
  ```
  git clone https://github.com/Jojole40990/Angry-Monkey.git
  ```
- Open Godot, click **Import**, and select the `project.godot` file
- Press **F5** or click the **Play** button to run the game

## Controls

| Action | Key |
|---|---|
| Move | WASD / Arrow Keys |
| Attack | Automatic |
| Pause | Escape |
